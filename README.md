# Portfolio de Youssef Missaoui

Portfolio personnel statique en `HTML / CSS / JavaScript`, conçu pour fonctionner directement en local et être hébergé facilement sur GitHub Pages.

## Fichiers du projet

- `index.html` : structure de la page
- `style.css` : style visuel et responsive
- `script.js` : menu mobile, animation d'apparition et navigation active
- `assets/` : images de projets et CV

## Lancer le portfolio en local

1. Ouvrir le dossier du projet.
2. Double-cliquer sur `index.html` ou l'ouvrir dans un navigateur.

Le site ne dépend d'aucun framework ni d'aucun serveur local.

## Personnalisation rapide

- Modifier les textes directement dans `index.html`
- Ajuster les couleurs depuis les variables CSS au début de `style.css`
- Remplacer les liens GitHub / LinkedIn / email / téléphone dans `index.html`
- Remplacer les images de projets dans `assets/`

Des commentaires simples ont été laissés dans `index.html` aux endroits utiles pour les remplacements.

## Hébergement sur GitHub Pages

1. Créer un dépôt GitHub.
2. Ajouter les fichiers du portfolio dans le dépôt.
3. Pousser le projet sur la branche principale.
4. Ouvrir `Settings > Pages` sur GitHub.
5. Dans `Build and deployment`, choisir `Deploy from a branch`.
6. Sélectionner la branche principale puis le dossier `/root`.
7. Enregistrer.

GitHub Pages générera ensuite une URL publique pour le portfolio.

## Remarque

Le portfolio a été conçu pour être sobre, professionnel, simple à modifier et compatible avec une ouverture directe du fichier `index.html`.
